AXLE: the Axiom Lean Engine ⚙️

Lean 증명 작업을 다루는 검증 도구 모음

AXLE는 Axiom이 공개한 Lean 증명 엔진으로, 대규모 형식 증명에 쓰인 핵심 인프라를 누구나 사용할 수 있게 꺼내 놓은 플랫폼입니다. Lean 코드에서 증명 검증, 오류 확인, 자동 복구, 정리 추출과 형태 변환을 빠르게 수행할 수 있습니다.

AxiomProver를 뒷받침한 Lean 메타프로그래밍 도구들을 바탕으로, 증명 조작을 부가 기능이 아니라 인프라 자체로 제공한다는 점이 핵심입니다.

공식 문서 웹 인터페이스 실습 데모 보기

AXLE가 하는 일

핵심은 Lean 증명 코드를 입력으로 받아, 맞는지 검사하고, 문제가 있으면 어디가 잘못됐는지 잡아내고, 필요하면 고쳐 보거나 구조를 바꾸는 것입니다. 빠른 검증, 안전한 실행, 그리고 증명 조작 유틸리티를 하나의 플랫폼으로 묶어 둔 형태라고 보면 됩니다.

검증

후보 증명이 실제로 Lean에서 통과하는지 확인

오류 확인

깨진 코드가 왜 실패하는지 바로 확인

자동 복구

실패한 증명을 다시 고쳐서 재검증 시도

구조 변환

정리를 쪼개거나 이름과 형태를 정리

대표 도구

verify_proof

후보 증명이 목표 정리를 만족하는지 검증

check

Lean 코드에 컴파일 오류가 있는지 검사

repair_proofs

깨진 증명을 자동으로 고쳐보고 다시 검증

extract_theorems

파일 안의 정리들을 분리해 추출

simplify_theorems

정리 구조를 더 단순한 형태로 정리

rename / normalize

선언 이름과 형식을 표준 형태로 정돈

더 많은 변환 도구는 공식 문서에서 확인할 수 있습니다.

실습에서 보게 될 흐름

문제 입력 증명할 정리
→
LLM 제안 증명 후보 생성
→
AXLE verify Lean 검증
→
Lean verified 통과 시 종료
실패하면
↓
AXLE repair 자동 수정 시도
→
피드백 전달 오류를 다음 시도에 반영
↺
다시 LLM 제안
